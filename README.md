# react 移动端项目
    一个招聘网站的wap端开发样例
# 项目相关
    1.下载依赖
    yarn install
    2.启动项目
    yarn start
    3.一般访问，目前只有登录注册
    http://localhost:8080/#/login
# mongodb安装
    启动
    C:\Program Files\MongoDB\Server\3.2\bin
    _> mongod
# 服务端项目下载 
    git clone git@github.com:liuer1211/react-serve.git
    1.下载依赖
    yarn install
    2.启动项目
    yarn start
